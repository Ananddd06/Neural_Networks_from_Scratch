# 🔥 Backpropagation with Cross-Entropy

## (3 Hidden Layers, **NO ZERO ACTIVATIONS**)

---

## 1️⃣ How we avoid zero activations (conceptually)

There are **3 valid ways**:

1. Choose **dummy values** so that
   $$
   Z > 0 \Rightarrow \text{ReLU}(Z) > 0
   $$
2. Use **positive bias**
3. Use **Leaky ReLU** (but you asked implicitly to keep ReLU behavior meaningful)

👉 For **manual math clarity**, we’ll use **ReLU with positive Z values**.

---

## 2️⃣ Network architecture

```
Input (2)
 → Hidden 1 (2, ReLU)
 → Hidden 2 (2, ReLU)
 → Hidden 3 (2, ReLU)
 → Output (2, Softmax)
 → Cross Entropy Loss
```

Single data point (so math is clean).

# 🔄 BACKPROPAGATION (STEP-BY-STEP)

---

## 🔥 Key identity (Softmax + CE)

$$
\boxed{
\frac{\partial L}{\partial Z_4} = \hat{y} - y

# 4️⃣ NUMPY IMPLEMENTATION (GUARANTEED NON-ZERO)

```python
import numpy as np

# --------------------
# Activations
# --------------------
def relu(Z):
    return np.maximum(0.01, Z)   # small floor to avoid exact zero

def relu_backward(dA, Z):
    dZ = dA.copy()
    dZ[Z <= 0] = 0
    return dZ

def softmax(Z):
    expZ = np.exp(Z - np.max(Z, axis=1, keepdims=True))
    return expZ / np.sum(expZ, axis=1, keepdims=True)

# --------------------
# Dummy data
# --------------------
X = np.array([[2.0, 1.0]])
Y = np.array([[0, 1]])

# --------------------
# Weights (positive init)
# --------------------
W1 = np.array([[0.5,0.2],[0.1,0.4]])
b1 = np.array([[0.1,0.1]])

W2 = np.array([[0.3,0.6],[0.4,0.2]])
b2 = np.array([[0.1,0.1]])

W3 = np.array([[0.2,0.5],[0.3,0.4]])
b3 = np.array([[0.1,0.1]])

W4 = np.array([[0.4,0.1],[0.2,0.3]])
b4 = np.zeros((1,2))

# --------------------
# Forward
# --------------------
Z1 = X @ W1 + b1
A1 = relu(Z1)

Z2 = A1 @ W2 + b2
A2 = relu(Z2)

Z3 = A2 @ W3 + b3
A3 = relu(Z3)

Z4 = A3 @ W4 + b4
Y_hat = softmax(Z4)

# --------------------
# Backward
# --------------------
dZ4 = Y_hat - Y
dW4 = A3.T @ dZ4
db4 = dZ4

dA3 = dZ4 @ W4.T
dZ3 = relu_backward(dA3, Z3)

dA2 = dZ3 @ W3.T
dZ2 = relu_backward(dA2, Z2)

dA1 = dZ2 @ W2.T
dZ1 = relu_backward(dA1, Z1)
```

## 5️⃣ Final takeaway (this is gold 🥇)

* Zero activations ⇒ **dead gradients**
* Positive bias + sane init ⇒ **healthy training**
* Softmax + Cross-Entropy ⇒ **clean gradients**
* Backprop is just **chain rule**, nothing mystical
